**BudgetsTab.tsx — 2 modifications:**

1. Supprimer `useProjectOverrides` et les appels `getPotentialAmount` / `setPotentialAmount`. Remplacer:
```typescript

  setPotentiel(n)
```

Supprimer les props `localProducts` et `onAddProduct` de BudgetsTab (plus nécessaires).
