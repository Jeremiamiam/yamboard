
2. Trois policies RLS Storage (le path suit la convention `{uid}/{clientId}/{filename}`):
```sql
CREATE POLICY "storage: owner upload"
  ON storage.objects FOR INSERT
  WITH CHECK (bucket_id = 'documents' AND auth.uid()::text = (storage.foldername(name))[1]);

CREATE POLICY "storage: owner read"
  ON storage.objects FOR SELECT
  USING (bucket_id = 'documents' AND auth.uid()::text = (storage.foldername(name))[1]);

CREATE POLICY "storage: owner delete"
  ON storage.objects FOR DELETE
  USING (bucket_id = 'documents' AND auth.uid()::text = (storage.foldername(name))[1]);
```

- `revalidatePath()` APRÈS la mutation, jamais avant
